Welcome to my interactive personal portfolio website! This modern, responsive portfolio showcases my journey as an AI/ML Developer and AI Tools Expert. Built entirely from scratch using cutting-edge web technologies and enhanced with AI assistance.

## 🌟 Overview

This portfolio website represents my professional journey, featuring my projects, technical skills, certifications, and educational background. The site offers an immersive experience with dynamic animations, interactive elements, and a sleek dark/light theme toggle.

### ✨ Key Features

- **🎨 Modern Design**: Clean, professional interface with neon-themed dark mode and Google-inspired light mode
- **📱 Fully Responsive**: Optimized for all devices - desktop, tablet, and mobile
- **🌙 Theme Toggle**: Seamless switching between dark and light themes
- **🎭 Interactive Animations**: Smooth transitions, particle effects, and scroll-triggered animations
- **📧 Contact Form**: Functional contact form with EmailJS integration
- **⚡ Performance Optimized**: Fast loading with optimized assets and animations
- **♿ Accessible**: Built with accessibility best practices

## 🚀 Quick Start

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for CDN resources)

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SanketKumarKar/my-portfolio.git
   cd my-portfolio
   ```

2. **Open locally:**
   ```bash
   # Option 1: Open directly in browser
   open index.html
   
   # Option 2: Use a local server (recommended)
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have live-server installed)
   npx live-server
   
   # Using PHP
   php -S localhost:8000
   ```

3. **View in browser:**
   Navigate to `http://localhost:8000` (if using local server) or simply open `index.html`

## 🎨 Customization

### Changing Themes
The website automatically adapts to your system preference, but you can manually toggle between:
- **Dark Mode**: Neon-themed with particle effects
- **Light Mode**: Google-inspired clean design

### Updating Content
1. **Personal Information**: Edit the content in `index.html`
2. **Styling**: Modify `styles.css` for core styles, `light-mode.css` for light theme
3. **Projects**: Update project cards in the HTML and corresponding images
4. **Particles**: Customize `particles.json` and `particles-light.json` for different effects
